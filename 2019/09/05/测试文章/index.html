继续往前走啦

发表于 2019-09-05 更新于 2019-09-08

  最近也开学一周了，学校里议论纷纷的是计算机学院研究生跳楼的事，开学第一天，从西十二五楼一跃而下，走前写满了对这个世界的控诉。这真的是多方面的因素造成的，有内心的敏感脆弱，不能和家人朋友的及时沟通，对研究生生活的误解等等。世界上没有真正地感同身受，我不能体会到他跳楼时的绝望。庆幸的是，在我伤心难过的时候，我身边总有人会来安慰我。

  我最难忘的人是我的化学老师陈冬。她从高二开始教我化学。那时候我的化学还可以吧（我个人觉得2333），高二的时候我身体不好，可能是学校刚装修完我们就住进去了吧，得了荨麻疹，去医院查了过敏源也没查出什么，具体症状就是一抓就红。当时还用这招逃过晚自习，第二天同学说化学老师找我订正作业，他们说我生病回家了哈哈哈。到了高三的时候，虽然又分了班，但还是化学老师教我，真的缘分。高三化学真的不好，印象里（经常）？考B吧？虽然离A就差几分，就很难受，我TM还是化学课代表！！！记得那年化学老师给过我一罐蜂蜜，“我儿子也吃这，增强抵抗力，给你带了一瓶”，这蜜的味道我现在也想不起来了（可能当时吃啥都没味道），到高考了还没吃完。有次找完化学老师订正试卷，化学老师就和我说压力别那么大啦，南大考不上，东南肯定能考上啊，就算考的再不好，那些没上过大学的不是一样活的好好的吗，放轻松~ 这话回头想想有点有趣，最后东南差了一分来了蜗壳。高考最后一天，考化学，感觉很糟糕，做的人心乱乱的，考完感觉不太好，出了考场就看到了化学老师，当时情绪还比较激动，抱了抱老师，就和她说：老师，我感觉好像要考C了。当时真的都快哭了。化学老师安慰了我，考完之后也没和大家一起happy，回家睡觉去了。到后来出成绩的时候，化学竟然是A，总分也还可以，很高兴地告诉了化学老师~后来还问了化学老师一些志愿的事情，这次回去见到了化学老师，四年了还是和之前一样年轻，老师家搬去了南京，说以后去南京的话请我们吃饭哈哈哈。

   我妈对我其实也蛮好的。我妈这人比较开明，虽然她有时候火气大让我不想理她。在我高三后期，基本上每天回家就摸手机，她也不怎么说我，我不止一次问过她，考的不好上了盐城工学院怎么办，她就说学校在那总是有人要去上的，你想上还能不给你上之类的，就是本三也给你填了。在我高考完之后，我是不想复读的，在出分的前几天，把高三的书基本上全卖给收废品的了，里面还有我整理的很好的数学错题集（sigh~）。在我想读研的时候，表示支持。后来侥幸保上了研，我不是很想读，我妈就是这样说的，你能读到哪算哪，你读不下去退学了，只要你开心，我们没有意见。做人要想通想开，但真的想通想开还是蛮难的。

  初中高中我身边的狐朋狗友还是蛮多的，也挺感谢他们的陪伴的。呆泽，还记得高二那晚路边的知足吧，

呆泽和我每年寒暑假都会一聚，每次的流程就是看电影，吃饭，逛聚龙湖，上网吧。每次聚会我们的标准结局就是在网吧。在FIFA三排的时候，呆泽经常骂我失位，吼我别拉后卫啊（2333）。呆泽是个属实的憨憨，这次回来呆泽和我说，我感觉我们的少年时光远去了。确实，同学们读研的读研，工作的工作，出国的出国，谈起未来眼里的星星也黯淡了。呆泽看着我们尘埃落定，也祝愿他考研顺利吧。呆璐是另一个憨憨，喜欢打游戏，感觉每天都很开心，在利物浦逆转巴萨晋级决赛之后怒删某绿色app,暑假回去之后他又装上了，真香~ 他就和我们吐槽，自从那个葛鸿宇喜欢上了巴萨，这个巴萨就不太行了2333。还有我们的弟弟，高三不想上课，让他爸跟老师请半天假，他爸问他半天够不够，要不然请一天吧。弟弟和我认识的时候还挺礼貌的，玩久了之后对我的态度变得和呆泽一样，对我FIFA里的失位表示不能忍受哈哈哈，希望在弟弟在厦门的这两年里可以去玩一圈~ 李憨憨已经在小盐城过上了开心令人羡慕的生活，陈老师也在扬大附中开始了教学生涯，晓哥去了香港，大玥开始在上海搬砖，顾博邰博开启磕盐生活，low总也在南京忙碌起来了，曾经的少年啊，真的开始要走向远方啦！“他喜爱北平，大概最大的原因是北平有几位说得来的朋友。” 我喜欢盐城，也是因为这里有我很多的回忆，但说得来的朋友，以后大部分也不会呆在盐城了。

  十年前，初一，国庆节语文老师说你们回去看阅兵吧，没什么作业；七年前，开开心心地上了高中；四年前，高考完来了新的城市武汉；现在又是一段旅程开始了，继续向前走吧，小强哥。
